# Python Moonlight Image

This Python Moonlight Image runtime supports many python versions and works with ARM and x86_64.

The Python Version is changeable for the user.

| Python Version | Architecture     |
| -------------- | ---------------- |
| 2.7            | ARM, x64, x86_64 |
| 3.7            | ARM, x64, x86_64 |
| 3.8            | ARM, x64, x86_64 |
| 3.9            | ARM, x64, x86_64 |
| 3.10           | ARM, x64, x86_64 |
| 3.11           | ARM, x64, x86_64 |
