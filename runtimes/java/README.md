# Java Moonlight Image

This Java Moonlight Image runtime supports many java versions and works with ARM and x86_64.

The Java Version is changeable for the user.

| Java Version | Architecture     |
| ------------ | ---------------- |
| 8            | ARM, x64, x86_64 |
| 11           | ARM, x64, x86_64 |
| 16           | ARM, x64, x86_64 |
| 17           | ARM, x64, x86_64 |
| 18           | ARM, x64, x86_64 |
| 19           | ARM, x64, x86_64 |
| 20           | ARM, x64, x86_64 |
| 21           | ARM, x64, x86_64 |
| 22           | ARM, x64, x86_64 |
